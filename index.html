What i know about you.

"What i know about you" é um site que demonstra tudo o que posso saber de você com um simples site com Javascript.

Javascript é a linguagem de programação padrão para se usar nos navegadores. Ela pode ser usada para melhorar a sua experiencia em um site, mas você já se perguntou as informações que o site pode obter de você só de entrar neles?

Este site demonstra isso, diversas informações sobre o seu computador obtidos apenas com o simples ato de entrar em um site.

Esse site usa o Javascript para pegar informações que qualquer outro site, com Javascript, consegue fazer.

Esse site irá pedir por algumas permissões, mas você não é obrigado a aceitar. Essas permissões irão ajudar a demonstrar ainda mais este estudo.

Esse site* NÃO envia as infomações aqui coletadas para fora do seu navegador. *O site está sendo hosteado no GitHub Pages, logo não posso confirmar que o GitHub não coleta informações. Para capturar o ip é usado uma API da Cloudfare, logo não posso confirmar que o Cloudfare não colete informações.

Clique em "MOSTRAR" para ver o que eu sei sobre você.
